## Key Features

- **Machine Learning Skin Analysis**
  - Upload a photo to receive personalized skin health insights.
- **Create a skincare routine**
  - Helps make skincare application more organized.
- **Secure Data Storage**
  - All user images are securely stored in the cloud.
- **Dark/Light Mode**
  - Customize your app's appearance to match your preference.
- **Best Product List**
  - Help users discover the best products tailored to their skin type.
- **Article List**
  - Help users to keep on update to newest skin care news.

## How To Use

1. Download the APK file from  [google drive](https://drive.google.com/uc?export=download&id=1m0LAkMiQKnDv_4j1-9pXWBEjtDpg0wIE) or you can download directly from GitHub sub Releases.
2. Enable installation from unknown sources on your Android device:
   - Go to **Settings > Security > Unknown Sources** and toggle it on.
3. Install the APK file and open the app.
4. Sign up or log in to access features.
5. Upload a clear picture of your skin for analysis.
6. View the analysis report and save it for future reference.

## Privacy and Security

Your data privacy is our top priority. All images and data are encrypted and processed securely in compliance with the latest standards. Data is not shared with third parties without your consent.

## Tools

This app was built using the following technologies:

- [Kotlin](https://kotlinlang.org/) for Android frontend development.
- [Android Studio](https://developer.android.com/studio/) as the IDE for Android development.
- [Google Cloud Platform](https://console.cloud.google.com/) as cloud provider.
- [TensorFlow](https://www.tensorflow.org/) for AI model development.
- [Flask](https://flask.palletsprojects.com/en/stable/) as the framework for serving AI models.
- [Gunicorn](https://gunicorn.org/) as the WSGI server for deploying AI models.
- [Node JS](https://nodejs.org/) for backend services.
- [Express.js](https://expressjs.com/) as the web application framework for the backend.
- [npm](https://docs.npmjs.com/about-npm) for managing backend dependencies.
- [Visual Studio Code](https://code.visualstudio.com/) as the code editor for backend development.
- [Postman](https://www.postman.com/) for API testing and development.
